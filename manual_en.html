User Manual

GestorOS — Professional Order Management System

1. Introduction

Welcome to the GestorOS system. This app is designed to manage orders from any phone in a professional and efficient way. It lets you handle tables, take orders, charge customers, print tickets, and generate smart reports.

The system is available in Spanish and English. Use the EN / ES button in the top-right corner to switch languages.

2. Login

1

Enter your authorized email and password on the welcome screen.

2

Tap Sign In. If credentials are correct, you will access the main orders dashboard.

💡 Only authorized staff can log in. If you forgot your password, contact the administrator.

3. Mobile Navigation

On phones, the screen is split into three panels accessed via the bottom navigation bar:

  • 🪑 Tables — list of open tables
  • 🍽 Menu — product catalog to add to the order
  • 🧾 Order — current order summary and charge button

The bottom bar hides automatically when you switch to History, Summary, or Settings, and comes back when you return to Orders.

4. Table Management

Opening a New Table

1

Tap the + New Table button in the tables panel.

2

Enter the table name or number (or pick a suggestion) and confirm.

3

The table appears in the list. When you open it, the system automatically tracks who is attending it.

Selecting an Existing Table

Tap any table in the list to view its current order or keep adding items.

⚠️ If someone else already has that table open on their phone, you will see a warning: "[Name] is editing this order. Continue anyway?" You can still proceed if needed.

Closing a Table Without Charging

Tap the ✕ next to the table name. If it has items, the system will ask for confirmation before deleting the order.

5. Taking Orders

1

With a table selected, go to the Menu panel. Use the search bar or browse by category.

2

Tap a product to open it. You can add a special note (e.g., "no onions") before confirming.

3

In the Order panel, adjust quantities with the + and − buttons, or tap the note icon (📝) to edit instructions for an item already added.

4

Optionally, type the customer's name in the order text field.

💡 The trash icon (🗑) removes an item from the order completely.

6. Charging and Payment

1

Tap the Charge button in the order panel. It is only available when there is at least one item.

2

Select the payment method: 💵 Cash or 💳 Card.

3

If paying by card, select the tip: 0 / 5 / 10 / 15 / 20 % or enter a custom amount.

4

Tap Confirm & Save. The sale is recorded, the table closes, and the history updates automatically.

5

If you have a Bluetooth printer connected, the option to print the ticket will appear.

💡 The system automatically records who charged each order. This is visible in History next to each sale (👤 Name).
⚠️ The button locks immediately when tapped to prevent duplicate charges. If something fails, it reactivates so you can try again.

7. History and Summary

Use the top tabs History and Daily Summary:

  • History: All orders within a date range. Each card shows the table, customer, time, payment method, and who charged it (👤 Name). You can reprint any ticket by tapping the 🖨️ icon.
  • Daily Summary: Total income, average ticket, breakdown by payment method, tips, and a ranking of best-selling products.
  • Export CSV: Download the data in Excel format.
  • Send by email: Send the report to the email address configured in ⚙ Settings.

8. Settings ⚙️

Your Name in Reports

Enter your name in the "Your name in reports" field and tap Save. This name will appear in History to identify who charged each order. Each person must set this on their own phone.

Report Destination Email

Enter the email address where History and Summary reports will be sent when using the "Send by email" button.

Bluetooth Printer

Tap Connect printer and select your Bluetooth device from the list. Once connected, the system remembers it for future tickets. To unlink it, tap Disconnect.

Menu Editor

Here you can manage the full catalog:

  • + New category — create a new section (e.g., "Desserts").
  • Rename — change the name of a category.
  • + Item — add a product with a name and price to that category.
  • Edit — modify the name or price of an existing product.
  • Delete — remove a category (and all its products) or an individual item.
💡 All menu changes sync automatically to all devices and to the public menu on the website.
⚠️ If the system cannot save a change (e.g., no internet), it automatically reverts to the previous state to avoid inconsistencies.
